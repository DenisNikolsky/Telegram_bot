# Телеграм-бот "Информация о субъекте"

Простой бот на `aiogram`, который отвечает на слово `Denis` (без учёта регистра) отправкой фото и информации о гуманоиде.

## Возможности

- Реагирует на `Denis`, `DENIS`, `denis` и т.д.
- Отправляет изображение (`AOzxTPn3z8Q.jpg`)
- Отправляет HTML-текст с данными

## Установка и запуск

1. Установите зависимости:
```bash
pip install aiogram==2.25.1
```
2. Получите токен бота у @BotFather и замените им значение переменной TOKEN в коде.
3. Поместите файл с изображением AOzxTPn3z8Q.jpg в ту же папку, что и скрипт. При необходимости измените имя файла в строке:

## Запуск
Выполните команду в терминале:

```bash
python bot.py
```

## Использование
Напишите боту в личные сообщения или в группу (если бот является администратором) слово Denis. Бот ответит фотографией, а затем текстом.
